Overview

Projects

The projects on this page are not meant as a finished catalogue, but rather as an overview of things that have occupied me over time and, in many cases, still do.

Some are already implemented, others are still forming. What they share is that they come from practical questions rather than from the urge to build something for its own sake.

SASD Website / CMS

Building a website and content structure with a deliberately simple and understandable technical approach.

Logging Framework

An own logging approach inspired by ideas such as log4j, with a focus on structure, flexibility and separation of concerns.

REST Services

Authentication, profile and project services built around clear responsibilities and a stable architecture.

Linux Test & Security Environment

A Linux-based environment for trying out security concepts, configurations and administration processes in practice.

Node.js / React Work

Smaller web projects that help connect modern frontend and backend approaches with more classical thinking.

Embedded / Electronics

Experiments and smaller projects around microcontrollers, hardware and the practical side of measurement and control.
