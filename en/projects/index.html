Selection

Projects

The projects on this page are not a finished catalogue, but rather an overview of things that have occupied me over time or still do.

Some are already implemented, others are still in progress or at an earlier stage. What they have in common is that they grew out of practical questions rather than out of the idea of building something just for the sake of it.

SASD website / CMS

Building a website with a deliberately simple and understandable technical approach. The focus is on clarity, maintainability and keeping content manageable without unnecessary dependencies.

Logging framework

Developing a logging approach inspired by ideas such as log4j while remaining flexible enough for different outputs and contexts.

REST services

A set of services for authentication, profile management and project organisation, with an emphasis on structure and clear responsibilities.

Linux test & security environment

A practical environment for exploring system administration, configuration and security-related questions in a Linux setting.

Node.js / React work

Smaller full-stack projects that help connect modern web approaches with more classical software thinking.

Embedded / electronics

Experiments and smaller projects with microcontrollers and hardware, where software meets physical systems.
