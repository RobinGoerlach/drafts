Link Collection

This is a selection of references, findings and pointers that have proved useful over time. Some of it is rather technical, some of it simply practical.

Not everything is new, and not every link will be equally interesting to everyone. But some things remain helpful for years, and that is exactly what this collection is for.

Linux & Administration

References around Linux, server operations, infrastructure and tools that remain useful in everyday work.

  • Debian – one of the Linux distributions I keep returning to, both in everyday use and on servers.
  • Apache HTTP Server – an important tool for a long time whenever classic web servers, configuration and stable deployment are involved.
  • Proxmox – useful for virtualisation, testing and the pragmatic operation of smaller infrastructures.

Software & Programming

Languages, tools and foundations that I keep returning to or that are useful for my own projects.

  • PHP – for web applications, services and pragmatic solutions that should remain understandable and maintainable.
  • JavaScript – for frontend work, small tools and everything that matters in the browser or modern web development.
  • C and C++ – especially interesting when things move closer to systems, tools, performance or hardware.
  • Java – a language that has accompanied me for a long time and remains relevant for many larger applications.

Data & Tools

Databases, analysis tools and other things that help turn information into something usable.

  • MySQL – a dependable tool for many web applications and small to medium-sized data solutions.
  • Oracle – interesting where more complex data handling, PL/SQL and larger environments are involved.
  • PostgreSQL – a database system that suits many modern applications and is often a good long-term choice.

Electronics & further findings

Links to hardware, microcontrollers and other things that connect theory and practice.

  • Arduino – a good starting point for practical work with microcontrollers, sensors and simple prototypes.
  • ESP32 – interesting for small connected systems, home automation and compact embedded projects.
  • SeaMonkey – not a tool for everyone, but a reminder that older and quieter software approaches can stay interesting much longer than expected.

And otherwise

This collection is not finished either. Some references will be replaced over time, others may stay useful longer than expected. So there will probably be more to add here as well.
