Link Collection

This is a selection of references, finds and pointers that have turned out to be useful over time. Some of them are technical, others are simply practical.

Not everything here is new, and not every link will matter equally to every visitor. Still, some resources remain helpful for years, and that is what this collection is for.

Profiles

A few public places where projects, professional traces and some older work can still be found.

  • GitHub – source code, experiments and a number of projects collected over time.
  • LinkedIn – the more conventional professional outside view for anyone looking for a structured overview.
  • Xing – not equally relevant everywhere anymore, but still part of the wider picture.

Linux & Administration

References related to Linux, server work, infrastructure and security tools that remain useful in everyday work.

  • The Linux Kernel Archives – the official source for the kernel itself and a good place to start when getting closer to Linux fundamentals.
  • Linux Manual Pages – dry at times, but often the most reliable place when commands and system details matter.
  • Linuxize – readable tutorials for administration work and common Linux tasks.
  • ArchWiki – useful even without running Arch Linux, because many explanations are clear and direct.
  • Apache HTTP Server Documentation – still a solid reference for classic web server configuration, modules and troubleshooting.

  • Proxmox – practical for virtualization, testing and the operation of small to medium environments.

Software & Programming

Languages, frameworks and documentation I keep returning to, or which are useful for my own projects.

  • SELFHTML – still a pleasant German-language reference for web fundamentals and many practical questions.
  • MDN Web Docs – often the first stop when it comes to solid documentation for HTML, CSS and JavaScript.
  • React Documentation – the current official documentation for component-based interfaces in the browser.
  • Node.js Documentation – reference material for server-side JavaScript, tooling and the runtime itself.
  • Tailwind CSS Documentation – useful when layouts and components should come together quickly while staying consistent.
  • DaisyUI Documentation – more of a component inspiration source than something every project has to depend on.

  • Docker Documentation – helpful for containers, testing environments and reproducible development setups.

Data & Databases

Databases and related tools that help turn information into something usable rather than simply stored.

  • MySQL Documentation – a long-standing reference for many web applications and small to medium data solutions.
  • PostgreSQL Documentation – useful for modern applications, more complex data models and clean SQL work.
  • Oracle Database – relevant when larger environments, PL/SQL and classic enterprise databases come into play.

Training & Learning

Platforms and course offerings that can be useful when a topic should be approached in a more systematic way.

  • LinkedIn Learning – a broad course catalog with a fairly classic continuing-education feel.
  • Udemy – useful when you want a quick first step into a very specific topic.
  • Microsoft Learn – the more sensible Microsoft entry point today; Channel 9 no longer exists in its old form.
  • Oracle Learning Library – labs and material around Oracle technologies.
  • Oracle LiveLabs – practical exercises for people who prefer trying things directly.
  • Oracle Dev Gym – short exercises and deeper practice, especially for SQL and PL/SQL.
  • Pluralsight – more technical in tone and often useful when a topic should be covered more thoroughly.
  • Coursera – a good fit when courses may lean a little more academic or university-backed.
  • edX Linux Courses – a useful addition for Linux topics in a more course-oriented format.
  • Linux Foundation Training – a good option when Linux and open source topics should be approached more formally and thoroughly.

Electronics & Other Finds

Links to hardware, microcontrollers and a few things that bridge the gap between theory and practice.

  • Arduino – a good place to start when working practically with microcontrollers, sensors and prototypes.
  • ESP32 / ESP-IDF – useful for small networked systems, home automation and compact embedded projects.
  • SeaMonkey – certainly not for everyone, but a nice reminder that quieter software approaches sometimes stay interesting longer than expected.

And otherwise

This collection is not finished either. Some references will be replaced over time, while others stay useful longer than expected. For international news sources, I would rather create a separate page or a clearly separated section later on, so that technical references and world media do not get mixed unnecessarily.
